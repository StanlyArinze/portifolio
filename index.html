Ao longo da minha trajetória acadêmica, adquiri uma sólida base teórica em diversas áreas da tecnologia. Desenvolvi diversos projetos academicos afim de colocar em prática toda experiência adquirida.


Atuei por quase 2 anos como desenvolvedor full stack na Lotus Hub, agência de publicidade, onde fui responsável pela administração da página e dos serviços digitais da empresa. Desenvolvi e implementei o site institucional, além de realizar manutenções e atualizações sempre que necessário. Essa experiência me proporcionou conhecimento prático em gestão digital, desenvolvimento web e suporte técnico, contribuindo diretamente para a presença online da agência


Tive também a oportunidade de estagiar na Lev Negócios, atuando como suporte operacional. Nesse período, desenvolvi habilidades como trabalho em equipe, comunicação clara e objetiva, agilidade no atendimento, flexibilidade e autonomia. Embora ainda não tenha experiência prática profissional diretamente na área de tecnologia, estou em busca de uma oportunidade para colocar meus conhecimentos em prática e crescer dentro de um ambiente desafiador e inovador. Tenho grande interesse em atuar em cargos de nível júnior ou programas de trainee, onde possa aprender com a equipe, contribuir com dedicação e me desenvolver continuamente como profissional.

Tecnologias e ferramentas que domino

CSS


Utilizado para estilizar a aparência de páginas web.

HTML


Linguagem de marcação usada para estruturar e dar significado ao conteúdo de uma página na internet.

PHP


Linguagem de programação embutida no HTML e serve para criar páginas dinâmicas, processar dados de formulários, gerenciar sessões, interagir com bancos de dados.

JavaScript


Linguagem de programação usada para criar interatividade em páginas web.

MySQL


Sistema de gerenciamento de banco de dados relacional, utilizado para armazenar, gerenciar e recuperar dados.

GitHub


GitHub é uma plataforma de hospedagem de código que utiliza o Git para controle de versão. O maior repositório de código do mundo.

Git


Um sistema de controle de versão distribuído que permite a desenvolvedores rastrear, gerenciar e colaborar em alterações de código-fonte.

Figma


Ferramenta de design de interface e prototipagem, utilizado para criar, colaborar e compartilhar layouts de sites, aplicativos e outros elementos visuais de forma totalmente online e colaborativa.

Power BI


Plataforma de Business Intelligence, serve para conectar, analisar e visualizar dados de diversas fontes, transformando informações complexas em relatórios e dashboards interativos.

NodeJs


Um ambiente de execução JavaScript de código aberto. Ele permite executar código JavaScript fora do navegador, geralmente no servidor back-end.

ReactJs


Biblioteca JavaScript de código aberto, usada para construir interfaces de usuário UI
